Hướng dẫn chạy frontend với vuejs
B1: Tải về Notejs và cài đặt
B2: Mở cửa sổ cmd với quyền admin gõ lệnh sau để cài đặt vue-cli
      npm install -g @vue/cli
B3: Vào folder frontend, mở cửa sổ cmd tại đây, gõ lệnh sau để khởi chạy project
      npm run serve
   (port đang để mặc định là 8081, chỉnh port trong package.json -> scripts -> serve : vue-cli-service serve --port 8081)
